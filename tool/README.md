## DSQLify ##

Created from OCaml 4.03 compiler frontend. 

### Building ###


1. `cd src/`
2. `./configure`
2. `make world`

### Testing ###
1. `./run.sh bankaccount`

A working installation of OCaml 4.03 or higher is assumed. In
particular, `ocamlc`, `ocamlopt`, `ocamllex`, `ocamlyacc` and
`ocamlde` tools are needed.
